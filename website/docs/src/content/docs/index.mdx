## Quick Start

### 1. Install

<Tabs syncKey="pkg">
  <TabItem label="npm">
    <Code code="npm install -g @dallay/agentsync" lang="bash" />
  </TabItem>
  <TabItem label="pnpm">
    <Code code="pnpm add -g @dallay/agentsync" lang="bash" />
  </TabItem>
  <TabItem label="Yarn">
    <Code code="yarn global add @dallay/agentsync" lang="bash" />
  </TabItem>
  <TabItem label="Bun">
    <Code code="bun i -g @dallay/agentsync" lang="bash" />
  </TabItem>
</Tabs>

> See the <a href="/guides/getting-started/#installation">Getting Started guide</a> for one-off execution (`npx`, `pnpm dlx`), binary downloads, and building from source.

### 2. Initialize and apply

<CommandTabs command="init" />
<CommandTabs command="apply" />

This documentation is the authoritative project reference.

> **Canonical Source**: All agent instructions and skills are maintained in `.agents/AGENTS.md`. This is the single source of truth—they get synced everywhere!

If you are deciding how your team should handle `.gitignore`, start with the <a href="/guides/gitignore-team-workflows/">Gitignore Team Workflows guide</a>. It explains the default managed mode, the committed-destination opt-out mode, and when `--no-gitignore` is appropriate.

If you use AgentSync from Windows, the <a href="/guides/windows-symlink-setup/">Windows Symlink Setup guide</a> covers native prerequisites, when WSL is the lower-friction option, and how to verify symlink health after `apply`.
